HTML:

HTML stands for Hyper Text Markup Language.HTML is the standard markup language for creating Web pages.
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the conten

PARAGRAPH TAG:

A paragraph always starts on a new line, and is usually a block of text.

FORMATTING TAGS:
  1. Bold:The bold tag is used in HTML to make text bold. It is written as
  2. Strong:The strong tag in HTML is used to indicate that a piece of text is strong importance, and is typically displayed in a bold font.
  3. Italic:The HTML i tag displays the text in italics.
  4. Emphasis:The emphasis tag emphasizes the text with semantic importance.
  5. Underlined Tag:It is used to indicate that a piece of text should be underlined.
  6. Subscript Tag:The subscript tag in HTML is used to define subscript text, which is text that appears lower than the normal text baseline.
  7. Superscript Tag:The superscript tag in HTML is used to define superscript text, which is text that appears higher than the normal text baseline
  8. Delete Tag:The delete tag in HTML is used to indicate that a piece of text has been deleted or removed.
  9. Mark Tag:The mark tag in HTML is used to highlight or mark important text, typically displayed with a yellow background color.
  10. Abbrevation Tag:The abbreviation tag in HTML is used to define an abbreviation or acronym, providing a way to specify the full expansion of the abbreviation.
  11. Blockquote Tag:The blockquote tag in HTML is used to indicate a quotation from another source, typically displayed with an indentation or a different font style.
  12. Preformatted Tag:The preformated tag in HTML is used to define preformatted text, which is displayed in a fixed-width font and preserves spaces, tabs, and line breaks.
  13. Quotation Tag:The quotation tag in HTML is used to define a short quotation, and is typically displayed with quotation marks around the text.
  14. Cite Tag:The cite tag in HTML is used to define the title of a work, such as a book, article, or movie, and is typically used to reference a source or citation.
  15. Address Tag:The address tag in HTML is used to define contact information, such as a physical address, email address, phone number, or other contact details.
  16. Code Tag:The code tag in HTML is used to define a fragment of code, such as a programming language or a script, and is typically displayed in a fixed-width font.
  17. Horizontal Tag:The hr tag in HTML is used to define a horizontal ruler, which is a horizontal line that separates sections of content.
  18. Break Tag:The br tag in HTML is used to define a line break, which starts a new line of text.
  19. Keyboard Tag:The kbd tag in HTML is used to define keyboard input, and is typically displayed in a monospace font.
BLOCK LEVEL ELEMENTS:

A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.A block-level element always takes up the full width available

INLINE ELEMENTS:

An inline element does not start on a new line. An inline element only takes up as much width as necessary.

TABLE TAGS:

HTML tables allow web developers to arrange data into rows and columns.A table in HTML consists of table cells inside rows and columns.
  • Table header(th):HTML tables can have headers for each column or row.Table headers are defined with th elements. Each th element represents a table cell.
  • Table row(tr):Each table row starts with a and ends with a tag.
  • Table colomn(td):Everything between and are the content of the table cell.
IMG TAG:

The HTML tag is used to embed an image in a web page.The tag has two required attributes:
  • src - Specifies the path to the image
  • alt - Specifies an alternate text for the image
ANCHOR TAG:

The HTML tag defines a hyperlink.The most important attribute of the element is the href attribute, which indicates the link's destination.The link text is the part that will be visible to the reader.
